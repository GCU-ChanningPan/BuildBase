# BuildBase — 電子黒板 / 工事写真管理 プロトタイプ

電子黒板（工事写真管理）のプロトタイプです。

## 起動方法（同僚向け）

[Node.js](https://nodejs.org/) がインストールされていれば、このフォルダで次のどちらかを実行するだけです（追加インストール不要）：

```sh
node start.js
# または
npm start
```

ローカルサーバーが立ち上がり、ブラウザでアプリが自動的に開きます（停止は Ctrl + C）。

> ⚠️ `電子黒板 工事写真管理 プロトタイプ.html` を**ダブルクリックで直接開くと真っ白**になります。
> .jsx を読み込む多ファイル版は `file://` ではブラウザのセキュリティ制限で動かないため、必ず上記スクリプト（`http://`）で起動してください。
> （配布用に1ファイルにまとめた `BuildBase_単一ファイル.html` はダブルクリックで開けます。）

## 構成

- `*.jsx` — 画面コンポーネント（iPad / iPhone / Web）
- `*.js` — データ・ストア
- `styles.css` — スタイル
- `電子黒板 工事写真管理 プロトタイプ.html` — エントリ HTML
- `assets/`, `uploads/` — 画像リソース

## 開発フロー（チーム）

このリポジトリは複数人で共同編集します。

1. 自分用のブランチを切る
   ```sh
   git switch -c feature/<your-name>-<内容>
   ```
2. コードを編集してコミット（コミットすると自動で GitHub に push されます）
   ```sh
   git add -A
   git commit -m "変更内容の説明"
   ```
3. GitHub 上で Pull Request を作成 → レビュー → `main` にマージ

> コミット後の自動 push は `.git/hooks/post-commit` で設定されています。
